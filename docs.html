OVERVIEW

Overview

Origin is a high-level, dynamically typed programming language designed for simplicity and expressiveness. It supports procedural and functional programming paradigms with a syntax inspired by Python, JavaScript, and C.

LEXICAL STRUCTURE

Lexical Structure

Tokens

Token Type Examples
KEYWORD let, const, if, elif, else, for, while, print, return, yield, break, continue, input, true, false, int, float, str, import, len, append
IDENTIFIER x, myVar, calculate
NUMBER 123, 3.14
ASSIGN / ASSIGN_OP =, +=, -=, *=, /=, %=, **=, //=
ARITH / COMP +, -, *, /, **, //, %, ==, !=, <=, >=, <, >
LOGIC and, or, not, &&, ||, !
UNARY ++, --
SPECIAL ??, ->, =>, <=>, ::

Comments

Single-line comments start with #.

# This is a comment
let x = 5
VARIABLES

Variables and Assignment

Declaration

let x = 10
const y = 20

Compound Assignment

Origin supports shorthand operations on variables and list indices.

x += 5
arr[0] -= 1
EXPRESSIONS

Expressions

Origin includes arithmetic, comparison, unary, logic, and special operators.

# Unary
++x
--y

# Logic
x and y or not z

# Special Operators
result = val ?? default
chain = a -> b => c
CONTROL FLOW

Control Flow

Conditionals

if x > 5 {
    print "big"
} elif x == 5 {
    print "equal"
} else {
    print "small"
}

Loops

Supports while and for...in range loops.

for i in range(1, 5) {
    if i == 3 { continue }
    print i
}

while x < 10 {
    x += 1
    if x == 7 { break }
}
FUNCTIONS

Functions and Calls

Functions are first-class objects. Supports return and yield.

result = myFunc(42)
return x + 5
yield x
LISTS

Lists & Indexing

let mylist = [1, 2, 3]
mylist[0] = 10
mylist.append(4)
print mylist

Lists support indexing, assignment, and the append() method.

LENGTH FUNCTION

len() Function

The len() function returns the number of elements in a list or string.

let mylist = [1, 2, 3]
print(len(mylist))  # 3

let mystr = "hello"
print(len(mystr))   # 5
INPUT / OUTPUT

Input / Output

x = input("Enter a number: ")
print x
