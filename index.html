Overview

Origin is a high-level, dynamically typed programming language designed for simplicity and expressiveness. It supports procedural and functional programming paradigms with a syntax inspired by Python, JavaScript, and C.

Comments

Single-line comments start with #.

# This is a comment
let x = 5

Variables and Assignment

Declaration

let x = 10
const y = 20

Compound Assignment

Origin supports shorthand operations on variables and list indices.

x += 5
arr[0] -= 1

Expressions

Origin includes standard arithmetic and comparison, as well as unique unary and special operators.

# Unary
++x
--y

# Special Operators
result = val ?? default
chain = a -> b => c

Control Flow

Loops

Supports while loops and for...in range expressions.

for i in range(1, 5) {
    if i == 3 { continue }
    print i
}

Functions and Calls

Functions are first-class objects. Origin supports both return for standard values and yield for generator-like behavior.

result = myFunc(42)
return x + 5
yield x
