Origin

A minimalist interpreted language built with a custom lexer, parser, AST, and execution engine.

Overview

Origin is designed for explicit structure, predictable execution, and full control over parsing and interpretation.

Control Flow

If / Elif / Else

if x > 10 {
  print "big"
} elif x > 5 {
  print "medium"
} else {
  print "small"
}

While Loop

let i = 0
while i < 5 {
  print i
  i = i + 1
}

For Loop

for i in range(0, 5) {
  print i
}
